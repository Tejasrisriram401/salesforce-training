# 🌩️ What is Salesforce?

Salesforce is a cloud-based CRM platform that helps organizations manage:

* Customers
* Sales
* Services
* Automation
* Data
* Applications

It provides powerful **low-code and no-code tools** to build business solutions quickly.

# 🏗️ Core Salesforce Concepts

## 🔹 App

A collection of tabs, objects, and tools designed for a specific business purpose.

### Examples:

* Sales App
* HR App
* Service App

---

## 🔹 Object

Objects are database tables used to store data.

### Standard Objects

* Account
* Contact
* Lead

### Custom Objects

* Property
* Employee
* Student

---

## 🔹 Tab

Tabs help users access objects and features easily through the UI.

Examples:

* Contact Tab
* Accounts Tab
* Reports Tab

---

# ⚙️ Configuration vs Coding

| Configuration             | Coding                        |
| ------------------------- | ----------------------------- |
| Click-based development   | Programming-based development |
| Fast and simple           | Flexible and advanced         |
| Uses Flow Builder & Setup | Uses Apex & LWC               |
| No-code / Low-code        | Full-code                     |

# 🧠 Important Salesforce Features Learned

## 🔄 Flow Builder

Used to automate workflows and reduce repetitive manual tasks.

### Example:

* Auto email sending
* Record updates
* Approval workflows

---

## 🤖 Prompt Builder & AI

Salesforce AI tools can:

* Generate emails
* Summarize information
* Improve customer communication

---

## 🔌 APIs

APIs help Salesforce communicate with external systems and applications.

Examples:

* Mobile Apps
* Third-party integrations
* External databases
